**代码介绍：**

stremlit_app_demo.py 为主文件

paramtuning.py 为调参函数

只需下载安装streamlit（version：1.12.0）

并在终端输入：

```python
python -m streamlit run streamlit_app_demo.py
```

就能进入到demo app



**数据介绍：**

为了确保业务数据安全，

本demo中的数据均为公开数据集。

***Churn_Modelling 数据集***  用于测试分类任务，**Exited**字段为类别标记。

***HousingData 数据集*** 用于测试回归任务，**MEDV**字段为标记。
